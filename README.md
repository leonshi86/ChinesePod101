# ChinesePod101 - 免费学中文网站

> 音频视频课程型语言学习网站

## 项目结构

```
ChinesePod101/
├── index.html                   # 首页
├── css/
│   └── style.css               # 样式
├── js/
│   └── main.js                 # 脚本
├── lessons/                     # 课程页面
│   ├── beginner/               # 初级课程
│   ├── intermediate/           # 中级课程
│   └── advanced/               # 高级课程
├── audio/                       # 音频课程
├── video/                       # 视频课程
├── resources/                   # 学习资源
│   ├── dictionary.html         # 词典
│   ├── pinyin-chart.html       # 拼音表
│   ├── grammar-guide.html      # 语法指南
│   └── survival-phrases.html   # 生存短语
├── pathways/                    # 学习路径
│   ├── survival-chinese.html   # 生存中文
│   ├── beginner-path.html      # 初级路径
│   └── hsk-prep.html           # HSK备考
└── assets/                      # 资源文件
    ├── images/
    └── fonts/
```

## 网站结构（模仿IndonesianPod101）

### 首页模块

| 模块 | 内容 |
|------|------|
| Hero区 | 大标题 + 开始学习按钮 |
| 课程类型 | 音频课程、视频课程、学习路径 |
| 功能介绍 | 为什么选择我们 |
| 学习路径 | 初级/中级/高级入口 |
| 免费资源 | 词典、拼音表、短语 |

### 课程页面结构

| 元素 | 说明 |
|------|------|
| 音频/视频播放器 | 主课程内容 |
| 课程描述 | 本课学什么 |
| 对话文本 | 中英对照 |
| 词汇表 | 生词列表 |
| 语法点 | 重点讲解 |
| 练习题 | 巩固练习 |
| 下载 | PDF/MP3下载 |

### 学习路径

| 路径 | 内容 |
|------|------|
| Survival Chinese | 生存中文（50课）|
| Beginner | 初级（100课）|
| Intermediate | 中级（100课）|
| Advanced | 高级（100课）|
| HSK Prep | HSK备考 |

---

## 核心功能

### 1. 音频课程
- 播放器界面
- 逐句播放
- 慢速/正常速度切换
- 文本同步高亮

### 2. 词汇学习
- 闪卡模式
- 发音按钮
- 例句展示

### 3. 学习进度
- 课程完成标记
- 学习统计

## 技术栈

- 纯静态网站（HTML/CSS/JS）
- 响应式设计
- 可部署到 GitHub Pages / Netlify
